# Formula Programs

Practice programs that take input from the user, apply a formula, and print the
result.

| File | What it calculates |
| --- | --- |
| `area_and_perimeter.c` | Area and perimeter of a rectangle, plus area and circumference of a circle. |
| `distance_converter.c` | Kilometers converted to meters, centimeters, inches, and feet. |
| `gross_salary.c` | Gross salary from basic salary, dearness allowance, and rent allowance. |
| `fahrenheit_to_celsius.c` | Fahrenheit to Celsius conversion. |
| `marks_percentage.c` | Total marks and percentage for five subjects. |
