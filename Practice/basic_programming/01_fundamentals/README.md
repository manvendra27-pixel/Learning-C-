# C Fundamentals

Small programs that introduce basic C concepts such as variables, arithmetic,
character values, user input, and standard library functions.

| File | What it demonstrates |
| --- | --- |
| `add_two_numbers.c` | Variable declaration, addition, and `printf()`. |
| `simple_interest_fixed_values.c` | Formula calculation with fixed values. |
| `simple_interest_user_input.c` | `scanf()` input and formula calculation. |
| `ascii_character_sum.c` | ASCII values stored in `char` variables. |
| `power_function.c` | Using `pow()` from `math.h`. |
